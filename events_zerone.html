Code Wars

Description

Code Wars is a tournament based event mixing the idea of ‘Duel’ and ‘Coding’ where two teams will be put against each other in order to qualify to the next round.

Team Size

2 members

Rounds

2 rounds

  • It will be in two days. One day will be best of 16 and best of 8 and next day will be best of 4 and best of 2.
  • Each round will be half hour with two questions given to each team.
  • Two teams competing will have same problem statement and the one to do it the first will win/proceed to the further round.
  • Problem statement will be a coding question and teams will be allowed to use any language they feel comfortable in.

Prize

First prize: 3000/-
Second prize: 2000/-

Timings

Online Round; 2nd Nov 6pm-8pm

Byte (Hackathon)

(In collaboration with CS Club)

Description

The participants are given 3 to 4 hours, so the problem statement is going to be a difficult task. There is only problem statement given to all teams. Internet access is allowed throughout the event.

Team Size

2 members

Criteria for Judging

  • Lines of code
  • Binary size
  • Processing time
  • Memory Utilization (not CPU utilization) For example, if they’re implementing a web server; The server shouldn’t crash given 1 million HTTP requests to the end point.

Example Scenario

If a submission ‘A’ takes 10 secs to compute the task and takes 500 mb of memory space. And another submission ‘B’ takes 15 secs to compute the task and takes 150 MB of memory space, then ‘B’ is the winner. 350 MB of extra memory is costlier than 5 seconds of extra time.

First prize: 2000/-
First prize: 1000/-
